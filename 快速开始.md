# 快速开始

## 1. 克隆项目后的初始化

```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，填入您的 GitHub 用户名
nano .env
# 或者用 VS Code 打开
code .env
```

## 2. 配置内容

编辑 `.env` 文件:

```bash
GITHUB_USERNAME=your_github_username
GITHUB_REGISTRY=ghcr.io
IMAGE_NAME=prometheus-alpine
```

## 3. 登录 GitHub Container Registry

```bash
docker login ghcr.io -u YOUR_USERNAME
# 输入 GitHub Personal Access Token 作为密码
```

## 4. 构建并推送镜像

```bash
# 使用脚本自动构建和推送
./build-push.sh
```

## 5. 验证推送成功

```bash
docker pull ghcr.io/YOUR_USERNAME/prometheus-alpine:latest
```

---

## 环境变量说明

| 变量              | 说明          | 默认值              |
| ----------------- | ------------- | ------------------- |
| `GITHUB_USERNAME` | GitHub 用户名 | 必需                |
| `GITHUB_REGISTRY` | 镜像仓库地址  | `ghcr.io`           |
| `IMAGE_NAME`      | 镜像名称      | `prometheus-alpine` |

## 常见问题

### Q: Token 存在哪里？

A: 使用 `docker login` 后会自动保存到本地凭证管理器（macOS 使用 osxkeychain）

### Q: .env 文件会被上传吗？

A: 不会，`.env` 已添加到 `.gitignore`

### Q: 如何重置 .env？

A:

```bash
rm .env
cp .env.example .env
# 重新编辑
```
